Blackwidow keyboard mapping
=

* F15 to F19 : Firefox, Sublime, Preview (PDF), Chrome, Safari
* Eject : Spotlight (OSX Builtin mapping)
* Keypad: 1 to 6 for tab 1-6; 7,8,9 for slate left/full/right;
          . for <cmd-tab>, - and + for tab prev/next;
          * for dictionary; 0 for iTerm; pad<enter>: Safari
          clear: Path Finder
          * git commit -m update: map to pad=
          * git commit -m "<cursor>": map to pad/
* F13/F14: Volume down/up
* Game mode: ON (Function keys are F1 to F12)
* Esc : via bw -> <super-,>  via system -> switch input source (Unicode hex/sogou)
* Caps: As another ctrl when hold; As Esc when short press;

<super-f> via service (automator: Global New Finder Window) mapped to : new finder
<super-n> : Evernote (China/yinxiang.com) : quick note
<super-l> : activate Safari
<super-j> : Activate QQ
<cmd-space> : Activate iTerm
<super-space> : Alfred 2
